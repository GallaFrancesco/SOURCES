Detailed Description

template<class CONV, class SUBSET, class STATE>
class BaseSubsetList< CONV, SUBSET, STATE >

Container for a list of subsets. For each subset the list of states is stored. Also, an inverse map (marking -> subset) is kept. Subset indices are stored in an internal zero-based way: however, external indices may be different. The function CONV.externalSubsetTo0BasedIndex() does the conversion.
