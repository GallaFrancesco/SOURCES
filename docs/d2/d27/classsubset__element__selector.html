Detailed Description

template<class T, class comparator = equal_to<T>>
class subset_element_selector< T, comparator >

Selector for a subset of elements, specified as a vector.
The subset is encoded in this way: a classes vector and a comparison value selClass are specified. An element k is selected if classes[k] == selClass.
