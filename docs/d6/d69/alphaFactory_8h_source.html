20 // Compute the alpha factors of a general distribution.
21 // @fg_expr the expression of the general distribution function f(x)
22 // @q the rate of the CTMC
23 // @accuracy requested accuracy for the truncation of the alpha-factors.
24 //
25 // Example of f(x) functions are the following:
26 // 3 * Exp(-0.2 * x)
27 // 0.4 * Pow(x, 2) * Exp(-0.5 * x) * Rect(0, 4)
28 // 0.3 * DiracDelta(5) + 0.7 * DiracDelta(10)
29 //
30 // Language elements include:
31 // 1.2345 Number term.
32 // +,-,*,/ Basic operands.
33 // Pow(t, e) Power of t with exponent e.
34 // Exp(t) Exponential of t
35 // Log(t) Natural logarithm of t
36 //
37 // and the two functions to specify a discontinuous impulse:
38 // DiracDelta(t) Deterministic Dirac impulse at time t
39 // I(t) Synonim for DiracDelta(t).
40 // Rect(a,b) Rectangular impulse of 1 in range (a,b)
41 //
42 // and the convenience (non-primitive) functions:
43 // Uniform(a,b) Uniform distribution in (a,b) = Rect(a,b)/(b-a)
44 // Triangular(a,b) Triangular distribution in (a,b), peak in (a+b)/2
45 // Erlang(l,r) Erlang distribution of rate l and r phases
46 // TruncatedExp(l,t) Exponential distribution of rate l truncated at time t
47 // Pareto(k,a) Pareto distribution
48 // Returns: a pair of vectors containing the alpha factors of f(x) and (1-F(x))
49 //
50 const std::vector<pair<double, double>>&
51 compute_alpha_factors_dbl(const char* fg_expr, const double q, const double accuracy);

54 // Verify that the @fg_expr text is a proper function for the computation of alpha factors.
55 // Verification includes:
56 // - the function is tested to be synctactically correct.
57 // - the area of the function is tested to be 1.0
58 // - the function is tested to be symbolically integrable

60 // Returns: nullptr if @fg_expr is a valid distribution for the computation of the
61 // alpha-factors. Returns an error message if one of the above tests
62 // fails.
