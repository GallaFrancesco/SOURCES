Detailed Description

Krylov-subspace methods for the solution of linear equation systems. These methods are taken from IML++ and modified to compile with boost c++ IML++ is a public domain library developed by NIST. It is a C++ templated library of modern iterative methods for solving both symmetric and nonsymmetric linear systems of equations.

Nonsymmetric matrix:

  • Is transpose available? -> Try QMR.
  • Otherwise, is storage at a premium? -> Try CGS or Bi-CGSTAB.
  • Otherwise try GMRES with a long restart.
